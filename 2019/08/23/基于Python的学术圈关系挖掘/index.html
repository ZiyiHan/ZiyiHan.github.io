基于Python的学术圈关系挖掘（本科毕业设计）

项目通过对百度学术中学者信息的关系爬取，针对学者关系进行分析，对学者之间的合作有一定意义。

结构设计

课题主要包括四个部分：数据集爬取，数据清洗，学者关系构建和数据可视化展示。

  • 数据集的获取：采用Python爬虫对百度学术的学者信息进行抓取，包括基础信息、论文资料以及合作学者信息

  • 数据清洗：清洗数据包括重复出现的姓名、中英文姓名，目的在于信息展示和合作次数统计

  • 学者关系构建：同校同事关系、非同校合作关系以及疑似师生关系

  • 数据可视化展示：学者信息可视化展示
